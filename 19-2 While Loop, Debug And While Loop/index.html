While Loop, Debug And Understand While Loop

While Loop

A while loop is used to repeat a specific blok of code until a condition is met. A real life example would be serving rice at a wedding repeatedly.

Structure of While Loop

while(condition)
{
//body of loop
}
while(condition): First condition is tested and if its true then body of the loop is executed.
//body of loop: A pair of parenthesis is a must if there is more than one statement in the body of while loop.

Why do we need while loop?

Suppose we want to print a message 5 times.

console.log('Hello');
console.log('Hello');
console.log('Hello');
console.log('Hello');
console.log('Hello');

This is the way to do it

var num = 0;
while(num < 5){
  console.log('Hello');
  num = num + 1;
}

How does while loop work?

  Code -------------------------- Status ----------------------------- Output

var num = 0; -------------------------------- num = 4 -------------------------------------- Hello
