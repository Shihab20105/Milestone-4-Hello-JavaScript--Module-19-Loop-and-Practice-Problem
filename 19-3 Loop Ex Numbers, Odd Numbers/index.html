Loop Example Numbers, Odd Numbers, Even Numbers

To write while loop you need 4 things

1: Loop Variable

2: Condition inside While

3: Loop Body

4: Change the Loop Variable

Example:

var number = 1; ------------------------------------------------------------- Loop Variable(1)
while(number <= 20){ -------------------------------------------------------- Condition inside While(2)
  console.log(number); ---------------------------------------- Loop Body(3)
  number = number + 2; ---------------------------------------- Change the Loop Variable(4)
}

roastGiven = 0;
while(roastGiven < 7){
  console.log('Give The Roast, Please !!' )
  console.log(roastGiven);
  roastGiven++;
}
