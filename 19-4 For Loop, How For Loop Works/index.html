For Loop, How For Loop Works, While Vs For Loop

For Loop

For Loop iterates through a block of code a number of times

for(expression 1; expression 2; expression 3)
{
   statement 1;
   statement 2;
}
1. variable declaration (expression 1;)
2. loop condition (expression 2;)
3. code (statement 1; and statement 2;)
4. increment/decrement (expression 3)

For Loop

Suppose you want print a massage using for loop

for(var i = 0; i < 5; i++){
   console.log('Hello');
}
Output:
Hello
Hello
Hello
Hello
Hello

How does for loop work?

    Code ------------------------------------------ Status ----------------------------- Output
for(var i = 0; i < 5; i++){ -------------------- i = 4 ----------------------------- Hello
   console.log('Hello'); --------------------------- i < 5 ----------------------------- Hello
} -------------------------------------------------- True ----------------------------- Hello
----------------------------------------------------------------------------------------- Hello
----------------------------------------------------------------------------------------- Hello
-------------------------------------------------- and when -----------------------------
-------------------------------------------------- i = 5 --------------------------------
-------------------------------------------------- i < 5 --------------------------------
--------------------------------------------------- False --------------------------------
--------------------------------------------- its becomes False --------------------------

Since condition evaluates to False, thus loop is terminated.

While Loop

var num = 0;
while(num < 5){
   console.log('Hello');
   num = num + 1;
}

* variable declaration (var num = 0;)

* loop condition (num < 5)

* code (console.log('Hello');)

* increment (num = num + 1;)

Output:
Hello
Hello
Hello
Hello
Hello
